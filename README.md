# cloud-skill

Skill 集合仓库

## 包含的 Skill

| Skill | 说明 | 安装方式 |
|---|---|---|
| [skill-distiller](skill-distiller/) | 帮小白把业务场景一步步变成可用的 Skill | `cp -r skill-distiller ~/.hermes/skills/` |

## 使用

```bash
# 克隆仓库
git clone https://github.com/Yunxj/cloud-skill.git

# 安装某个 Skill（以 skill-distiller 为例）
cp -r cloud-skill/skill-distiller ~/.hermes/skills/

# 以后更新
cd cloud-skill && git pull
```
