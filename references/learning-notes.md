## 2026-04-16 | 空格丶 9篇公众号文章合集
**资料**：zephyrwang6 公众号 9 篇 Skill 方法论文章（github.com/zephyrwang6/myskill）

**学到了什么**：
- **方法论**：5种创建 Skill 的方法（反推法/流程法/归纳法/经验提炼法/职业拆解法），都从已有经验出发
- **公式**：提示词 + API + 自动化流程 = 批量出图
- **迭代闭环**：创造 → 记录使用（skill-logger）→ 分析数据（skill-analyzer）→ 高频打磨/低频淘汰
- **管理方法**：全局目录 + 软链接 + GitHub 同步 + 分类仓库 + 环境变量管理 API Key
- **自动化标准**：连续 20 次不需要手动改 = 可以全自动
- **系统化思维**：单独是零件，串联是系统；Skill 正在吃掉 SaaS
- **生态**：ClawHub 平台 10,322 个 Skill，6 大领域（办公协同 35%/信息研究 20%/开发设施 15%/运维优化 15%/多媒体 10%/垂直场景 5%）
- **好话术**："Skill 是零件，系统是机器"、"脑子里的东西挪到 Skill 里就变成了可复用的能力"、"真正有用的 Skill 不是凭空想象出来的，而是从你每天重复做的事情中提炼出来的"
